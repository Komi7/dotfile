#### setup
```
git clone https://github.com/komi7/dotfile.git
cd dotfile
sudo ./install.sh
```

#### (extra)
```
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```

#### active display manager
```
sudo systemctl enable sddm
```
